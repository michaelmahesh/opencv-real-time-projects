# OpenCV Real-Time Projects (Anaconda + Jupyter)

A collection of **real-time computer vision projects** built using  
**Python, OpenCV, MediaPipe, and Deep Learning**, executed in  
**Jupyter Notebook (Anaconda Environment)**.

This repository is ideal for:
- 🎓 Students
- 👨‍🏫 Faculty / Trainers
- 🤖 AI & Computer Vision Enthusiasts

---

## 📌 Projects Included

✅ Brightness Control with Hand Detection  
✅ Real-Time Face Mesh Detection (MediaPipe)  
✅ Multi-Face Recognition System  
✅ Live Webcam Drawing Canvas  
✅ Finger Counter using Computer Vision  
✅ Car Detection in Video Streams  
✅ Gun Detection using Deep Learning  
✅ Invisible Cloak (Invisible T-shirt Effect)  
✅ License Plate Recognition System  
✅ Left & Right Hand Detection  
✅ Virtual Mouse using Hand Gestures
